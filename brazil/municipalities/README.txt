
This data set matches identificators and names of Brazilian municipalities across various Brazilian institutions (IBGE, TSE, STN, and Comarcas).

//-----------------------------------------------------------------//
// This match solves the following problems.
// - No single municipality identificator across IBGE, TSE and STN.
// - No publicly available IDs for comarcas (judicial districts). 
// - Ids and name strings change across years in both institutions.
// - Municipalities are created or cease to exist across years.
//-----------------------------------------------------------------//


//-----------------------------------------------------------------//
// Observations
// - Data provided in .dta and .csv formats.
// - Updated until 2018, so use with care for posterior years.
//-----------------------------------------------------------------//
